# .revalue

Convert a value to another value through a pre-map

globals

    {
        red: orange
    }

property

    {
        color: {
            red: #880000
        }

        background: {
            red: linear-gradient(red, #880000)
        }
    }

entity

    {
        p: {
            color: {
                red: 'color.red.reshade+2'
            }
        }

        strong: {
            color: {
                red: 'p.color.red.reshade-2'
            }
        }
    }
